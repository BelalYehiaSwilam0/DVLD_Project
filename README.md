# 🚦 Driving License Management System (DVDL)

## Overview
**DVDL** is a **Desktop Application** for managing driving licenses using **3-Tier Architecture**:

- **Presentation Layer**: Windows Forms UI for user interaction  
- **Business Layer**: Implements logic, rules & validation  
- **Data Access Layer**: Handles SQL Server operations via **Stored Procedures** using **ADO.NET**

The system supports **local and international licenses**, tracks tests and conditions, and ensures data accuracy and integrity.

---

## Key Features
- Complete workflow: registration → testing → license issuance  
- Clear separation of layers for maintainable, scalable code  
- Secure user authentication with hashed passwords  
- Efficient SQL Server operations via **T-SQL Stored Procedures**  
- Handles multiple license types, test statuses, and specific conditions
